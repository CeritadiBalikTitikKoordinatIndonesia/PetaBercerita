# PetaBercerita

Buku "Mengungkap Cerita di Balik Titik Koordinat Indonesia"

Miles Tones:
1. Mengumpulkan Materi Tentang Titik Koordinat, dan SIG:
a. Titik Koordinat
b. SIG
c. Cerita apa yang bisa muncul dari sebuah titik koordinat

2. Dasar Geospasial untuk Pemula:
a. Apa itu geospasial
b. Tipe data geospasial (Point, Line, Polygon)
c. Format data GeoJSON
d. Apa itu koordinat (lat, lon) dan cara membacanya

3. Dasar-Dasar JavaScript untuk Geospasial:
a. Mengelola data JSON
b. Mengambil data menggunakan fetch()
c. Memproses array lokasi & menambah informasi cerita
d. Struktur folder proyek peta

4. Pengenalan LeafletJS:
a. Apa itu LeafletJS
b. Cara membuat peta dasar
c. Menambahkan marker
d. Layer & tile layer

5. Menampilkan Titik Koordinat Indonesia Secara Interaktif
a. Dataset titik: kota, fenomena alam, lokasi sejarah, budaya
b. Marker warna berbeda berdasarkan kategori cerita
c. Cluster marker (Leaflet.markercluster)
d. Membuat tooltip “cerita singkat”

6. Mengolah Data Menjadi GeoJSON di JavaScript
a. Mengubah data koordinat mentah (CSV/JSON) → GeoJSON
b. Struktur GeoJSON
c. Cara menampilkan GeoJSON di Leaflet
d. GeoJSON untuk batas wilayah: provinsi, kabupaten, kecamatan
e. Menambahkan cerita pada setiap wilayah (klik polygon)
